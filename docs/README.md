# GitHub Pages Demo

Ta mapa vsebuje javni demo oglednika nadzornih plošč. Odpri `index.html` na GitHub Pages.

## Uporaba
- V polje “Manifest JSON URL (Raw)” prilepite URL do vašega `dashboards/manifest.json`.
- Kliknite “Naloži manifest”.
- Iz levega seznama izberite ploščo; desno se nariše graf (Chart.js).

## Privzeti manifest
- Trenutno je prednastavljen: `https://raw.githubusercontent.com/robertpezdirc-eng/copy-of-copy-of-omniscient-ai-platform/feat/20-dashboards/dashboards/manifest.json`
- Po združitvi v `main` posodobite URL, če se veja spremeni.
